Instrucciones para Jugar

  1. Crear un Nuevo Juego:
    • Ruta: /game/create
    • Descripción: Esta ruta reinicia el juego y lo prepara para una nueva partida.
  2. Posicionar los Barcos:
    • Selecciona el jugador que posicionará los barcos
    • Introduce las posiciones de tus barcos en formato json, sigue el ejemplo, para indicar las coordenadas
    • Descripción: Esta tabla coloca los barcos para el jugador especificado. Envía un JSON con las posiciones de los barcos.
    • Ejemplo de JSON para input en el siguiente orden:
      1. Porta-aviones
      2. Crucero 1
      3. Crucero 2
      4. Destructor 1
      5. Destructor 2
      6. Destructor 3
      7. Submarino 1
      8. Submarino 2
      9. Submarino 3
      10. Submarino 4
                              
                        [
                            [{"x": 0, "y": 0}, {"x": 0, "y": 1}, {"x": 0, "y": 2}, {"x": 0, "y": 3}, {"x": 0, "y": 4}], 
                            [{"x": 1, "y": 0}, {"x": 1, "y": 1}, {"x": 1, "y": 2}, {"x": 1, "y": 3}],
                            [{"x": 2, "y": 0}, {"x": 2, "y": 1}, {"x": 2, "y": 2}, {"x": 2, "y": 3}],
                            [{"x": 3, "y": 0}, {"x": 3, "y": 1}, {"x": 3, "y": 2}],
                            [{"x": 4, "y": 0}, {"x": 4, "y": 1}, {"x": 4, "y": 2}],
                            [{"x": 5, "y": 0}, {"x": 5, "y": 1}, {"x": 5, "y": 2}],
                            [{"x": 6, "y": 0}, {"x": 6, "y": 1}],
                            [{"x": 7, "y": 0}, {"x": 7, "y": 1}],
                            [{"x": 8, "y": 0}, {"x": 8, "y": 1}],
                            [{"x": 9, "y": 0}, {"x": 9, "y": 1}]
                        ]
  3. Ver el Estado del Juego:
    • Ruta: /game/status
    • Descripción: Esta ruta devuelve el estado actual del juego, incluyendo los tableros de ambos jugadores. 0 - indica sin barco, 1 - indica hay barco, y 2 - indica que fue atacado
  4. Ver el Resumen del Jugador:
    • Ruta: /player/:playerNumber
    • Descripción: Esta ruta devuelve el tablero del jugador especificado.
